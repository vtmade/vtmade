## Overview

Hanami is a generative art project exploring data as an aesthetic medium. Where code meets creativity.

## Concept

The project draws inspiration from the Japanese tradition of flower viewing (hanami), translating the beauty of natural patterns into algorithmic art.

## Features

- Generative algorithms
- Data-driven aesthetics
- Code-based art creation
- Visual pattern exploration
